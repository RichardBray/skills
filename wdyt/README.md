# wdyt

Give an honest opinion without implementing anything, triggered when a message ends with "wdyt" or "what do you think".

## Installation

```sh
npx skills add https://github.com/RichardBray/skills --skill wdyt
```

## Usage

Just end a message with "wdyt" or "what do you think":

```
Should we move auth to middleware instead? wdyt
```

Takes a real position, surfaces concerns and tradeoffs, asks clarifying questions, and waits for a response before writing any code or plan.
