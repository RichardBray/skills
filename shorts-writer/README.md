# shorts-writer

Write short video scripts for a developer-focused tech shorts channel.

## Installation

```sh
npx skills add https://github.com/RichardBray/skills --skill shorts-writer
```

## Usage

```
/shorts-writer [topic]
```

Generates a punchy, lowercase script with a hook that subverts assumptions, an honest take on limitations, and a "subscribe for more" sign-off. Scripts are saved as `.md` files in a new directory named after the topic.
