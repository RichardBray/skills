# longs-writer

Write long-form YouTube scripts (~8 min) for a dev-focused tech channel.

## Installation

```sh
npx skills add https://github.com/RichardBray/skills --skill longs-writer
```

## Usage

```
/longs-writer [topic]
```

Produces a structured script (title options, intro, explanation, demo bullet points, outro, sources) written section by section with user feedback between each. Style is casual, lowercase, one thought per line. Scripts are saved as `.md` files in a new directory named after the topic.
