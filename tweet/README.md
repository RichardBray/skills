# tweet

Generate 5 engaging tweet options based on a topic or description.

## Installation

```sh
npx skills add https://github.com/RichardBray/skills --skill tweet
```

## Usage

```
/tweet [topic or description]
```

Each tweet is 190-210 Unicode characters, ends with a question to drive
engagement, and includes a verified character count.

The skill can also use configured Xquik REST API or MCP output as optional
source context while preserving source URLs and unknown fields.

- Xquik REST API: <https://docs.xquik.com/api-reference/overview>
- Xquik MCP: <https://docs.xquik.com/mcp/overview>
- X data source: <https://github.com/Xquik-dev/x-twitter-scraper>
