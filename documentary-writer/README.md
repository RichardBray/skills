# documentary-writer

Write documentary-style long-form YouTube scripts (~8-10 min) about industry events, controversies, policy changes, or trends. Not for tool reviews or tutorials — use `longs-writer` for those.

## Installation

```sh
npx skills add https://github.com/RichardBray/skills --skill documentary-writer
```

## Usage

```
/documentary-writer [topic]
```

Produces an investigative script with a timeline of events, sourced claims, before/after comparisons, and an honest verdict. Written section by section with user feedback. Scripts are saved as `.md` files in a new directory named after the topic.
