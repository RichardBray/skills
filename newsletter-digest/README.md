# newsletter-digest

Fetch developer newsletter RSS feeds and Hacker News top stories, then curate the 10 best items as YouTube video topic ideas.

## Installation

```sh
npx skills add https://github.com/RichardBray/skills --skill newsletter-digest
```

## Usage

```
/newsletter-digest
```

Default feeds: JavaScript Weekly, React Status, Node Weekly, and Daily Dev. Also pulls from Hacker News top stories. Each pick includes a title, URL, source, and a one-sentence YouTube video angle.
